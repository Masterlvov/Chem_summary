# 高分子知识


#### 分子链
- 分子量大小等效于整条聚合物链有多“长”（重复单元数量）（比如 n=1000 还是 n=10000）	与强度、熔体粘度、韧性相关：分子量越大，链越长，缠结越多 → 更强更韧，但更难加工。
- 支链多少/长短即主链上有没有“分叉”，分叉有多长，与	结晶度、密度、透明性相关：支链越多越长 → 越难结晶 → 越软越透明。
#### 怎么调控使得各类基团（如甲基）全部在同侧，线性无支链？
- 其中，带甲基的碳是一个手性中心。如果每个单元的甲基都在主链同一侧（如都朝上），分子链就能规则折叠、结晶 → 得到高强度、高熔点（～165°C）的塑料。
- 等规PP（isotactic）：甲基全同侧 → 可结晶 → 实用材料；
- 无规PP（atactic）：甲基随机分布 → 无法结晶 → 粘稠橡胶状，无结构强度。
- 使用立体选择性催化剂，在聚合时“指挥”丙烯单体以固定方向插入链中。Ziegler-Natta 催化剂（主催化剂：TiCl₃ 或 TiCl₄（负载在 MgCl₂ 上）助催化剂：Al(C₂H₅)₃（三乙基铝）工作原理：Ti 活性中心具有固定的空间结构；丙烯单体只能从特定方向靠近并插入增长链；每次插入都保持相同取向 → 甲基始终在同侧。等规度（isotacticity）可达 90–98%；分子链线性、无支链（丙烯本身是α-烯烃，配位插入机理，链增长在金属中心有序进行；聚合不产生支化））
